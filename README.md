# 🏥 Hospital Emergency Room Dashboard

An interactive **Power BI dashboard** built to analyze Emergency Room (ER) operations and provide actionable healthcare insights. This dashboard helps monitor patient flow, waiting time, admission status, referrals, demographics, and overall hospital performance through interactive visualizations.

---

## 📌 Project Overview

The Hospital Emergency Room Dashboard transforms raw healthcare data into meaningful insights for hospital administrators and decision-makers. It enables users to monitor key performance indicators, identify operational bottlenecks, and improve patient care through data-driven decision-making.

---

## 🎯 Objectives

- Monitor emergency room performance.
- Analyze patient waiting time and satisfaction.
- Track admission and referral trends.
- Understand patient demographics.
- Identify peak patient hours and busiest days.
- Support operational and strategic healthcare decisions.
